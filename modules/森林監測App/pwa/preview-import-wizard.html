Excel 匯入 Wizard 雛形

此頁面用於獨立預覽 5 步驟 wizard 流程。不會連線 Firebase、不會寫入任何資料，最後一步僅輸出 DRY-RUN payload 到瀏覽器 console。

建議測試檔案：
H:\我的雲端硬碟\cct\林業技師事務所\標案\024 中華紙漿股份有限公司\森林經營計畫書撰寫\樣區調查\調查紀錄_0408.xlsx
（19 樣區、930 株、44 種樹種、6 種狀態碼）

5 步驟流程

  1. 選檔 — SheetJS 解析 .xlsx，自動偵測哪些工作表是樣區明細
  2. 樣區對應 — 勾選要匯入的工作表，指定 plot code
  3. 欄位對應 — 把 Excel 欄位對應到 MRV 目標欄位（自動猜測）
  4. 樹種與代碼 — 列出未知樹種、設定狀態碼對照
  5. 預覽 DRY-RUN — 顯示將寫入結構，不實際寫入

✏️ 此預覽用 preview-app-mock.js 取代真實 js/app.js，wizard 程式邏輯與正式環境完全相同（複製自 js/import-wizard.js）。
