## 專案定位

本專案用於：
- 森林經營計畫書（FMP）撰寫與審查
- 碳匯專案文件（AR-TMS0001 / 0002 / 0003 / 0004）
- 林業永續輔導計畫
- 林業政策與技術報告
- GIS 與空間分析（圖層裁切、套疊、坡度分析、變遷偵測）

所有輸出須符合：
- 台灣林業實務
- 林業及自然保育署規範
- IPCC 2006 Guidelines（LULUCF）
- MRV（測量、報告、查證）原則
- TACCC（透明、一致、可比、完整、準確）

## 工作注意事項

### 林業專業
- 座標系統一律 **TWD97 TM2**
- 物種引用前必須先查 `species-conservation-lookup` 確認保育等級
- 碳匯計算須符合 IPCC LULUCF + 對應 TMS 方法學
- 章節層級格式：壹、 → 一、 →（一）→ 1. →（1）

### 資料安全
- **`.claude/` 一定要排除**（含 token 的本地設定，絕不上雲）
- 樣區資料若涉及私有林地，需去識別化
- API Key 不寫死在程式碼

### 收工同步
- 收工前說「收工」讓 Claude 同步三方
- commit 訊息要寫清楚：做了什麼 + 為什麼（不要寫「更新」「修改」）
